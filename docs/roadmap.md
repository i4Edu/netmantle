# Roadmap

This repository implements **Phase 0** (project foundation) plus the bare
bones of **Phase 1** (MVP backup). Subsequent phases — change notifications,
auditing/search, compliance, discovery, push automation, in-app CLI,
distributed pollers, runtime state, multi-tenancy/HA, and hardening — are
tracked in the project plan and will land as focused follow-up PRs.

| Phase | Theme | Status |
| ----- | ----- | ------ |
| 0 | Project foundation | ✅ in this repo |
| 1 | MVP backup (inventory, SSH, Cisco IOS, git versioning, basic UI) | 🚧 partial in this repo |
| 2 | Change management & notifications | ⏳ planned |
| 3 | Auditing & search | ⏳ planned |
| 4 | Configuration compliance | ⏳ planned |
| 5 | Discovery & NMS sync | ⏳ planned |
| 6 | Push / pull automation | ⏳ planned |
| 7 | In-app CLI & distributed pollers | ⏳ planned |
| 8 | Runtime state auditing & compliance | ⏳ planned |
| 9 | Multi-tenancy & HA | ⏳ planned |
| 10 | Hardening, NETCONF/RESTCONF/gNMI, topology, GitOps mirror | ⏳ planned |
